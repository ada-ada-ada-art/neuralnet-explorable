Neural Network Explorable Explanations

This is the index for the three iterations of
the Neural Network Explorable Explanation project by Jesper Hyldahl Fogh.

Click the titles below to read a short intro to each iteration, and to show the link to the prototypes.
The prototypes are best experienced in either Firefox or Google Chrome.

The Visualized Network

This prototype does not represent a finished explorable explanation.

This version instead represents the most complete of the versions of this iteration. However this one was still not a complete experience.
